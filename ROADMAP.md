# Дорожная карта развития Sclon

> **Sclon** — консольное .NET-приложение для склонения русских фраз по шести падежам.
> Текущее состояние: базовое склонение существительных (1–3 скл.) и прилагательных,
> предлоги/союзы, беглые гласные (~20 слов), одушевлённость.
>
> **Как пользоваться:** отмечай `[x]` по мере выполнения. `[/]` — в работе, `[ ]` — не начато.

---

## Этап 1. Фундаментальные исправления

### 1.1 Местоимения

Притяжательные (`мой`, `твой`, `свой`, `наш`, `ваш`), личные (`он`, `она`, `оно`, `они`, `я`, `ты`, `мы`, `вы`),
указательные (`этот`, `тот`, `такой`, `таков`), возвратное (`себя`), определительные (`сам`, `весь`, `каждый`).

Сейчас «мой старый друг» → «моя старого друга» (местоимение не склоняется, анализатор падает в Noun).

**Файлы:** `Models.cs`, `RussianAnalyzer.cs`, `RussianDeclensionEngine.cs` | **Оценка:** ~1 день

- [ ] Добавить `Pronoun` в `PartOfSpeech` (`Models.cs`)
- [ ] Склонение притяжательных: `мой`, `твой`, `свой`, `наш`, `ваш`
- [ ] Склонение указательных: `этот`, `тот`, `такой`, `таков`
- [ ] Склонение личных: `я`, `ты`, `он`, `она`, `оно`, `мы`, `вы`, `они`
- [ ] Склонение возвратного: `себя`
- [ ] Склонение определительных: `сам`, `весь`, `каждый`, `любой`
- [ ] Диспетчеризация в `DeclinePhrase` для `Pronoun`

### 1.2 Числительные

Количественные (`один`, `два`, `три`, `четыре`, `пять`…) и порядковые (`первый`, `второй`, `третий`…).

Сейчас «один дом», «два кота» — числительные не склоняются, порядковые могут путаться с прилагательными.

**Файлы:** `Models.cs`, `RussianAnalyzer.cs`, `RussianDeclensionEngine.cs` | **Оценка:** ~1 день

- [ ] Добавить `Numeral` в `PartOfSpeech` (`Models.cs`)
- [ ] Склонение количественных: `один` — `два` — `три` — `четыре` — `пять`… (согласование)
- [ ] Склонение порядковых как прилагательных: `первый`, `второй`, `третий`
- [ ] Склонение собирательных: `двое`, `трое`…
- [ ] Диспетчеризация в `DeclinePhrase` для `Numeral`

## Этап 2. Ядро русской морфологии

### 2.1 Множественное число

Сейчас все слова считаются единственным числом. Нужна полноценная поддержка pluralia.

**Файлы:** `RussianAnalyzer.cs`, `RussianDeclensionEngine.cs` | **Оценка:** ~2–3 дня

- [ ] Определение числа в `RussianAnalyzer` (по окончанию/словарю)
- [ ] Таблицы plural для 1-го склонения (м.р. тв./мяг./шип.)
- [ ] Таблицы plural для 2-го склонения (ж.р. на -а/-я)
- [ ] Таблицы plural для 3-го склонения (ж.р. на -ь)
- [ ] Таблицы plural для ср.р. (на -о/-е)
- [ ] Plural разносклоняемых (на -мя → имена)
- [ ] Plural прилагательных (тв./мяг./шип.)
- [ ] Согласование прил. + сущ. по числу в `DeclinePhrase`

### 2.2 Расширение словаря беглых гласных

Текущий словарь (~20 слов). Цель — ~100 наиболее употребительных слов с беглой гласной.

**Файл:** `RussianAnalyzer.cs` | **Оценка:** ~2 часа

- [x] Текущий базовый набор: `пёс`, `день`, `пень`, `огонь`, `камень`, `корень`, `котёл`, `рот`, `лоб`, `лёд`, `сон`, `лев`, `потолок`, `угол`, `ветер`, `парень`, `боец`, `молодец`, `купец`, `отец`, `певец`

Добавить:

- [ ] `ковёр → ковр`, `замок → замк`, `узел → узл`
- [ ] `платок → платк`, `носок → носк`, `молоток → молотк`
- [ ] `огонёк → огоньк`, `кошелёк → кошельк`, `ремешок → ремешк`
- [ ] `чулок → чулк`, `мешок → мешк`, `гребень → гребн`
- [ ] `стебель → стебл`, `кисель → кисел`, `шатёр → шатр`
- [ ] `жеребец → жеребц`, `гонец → гонц`, `дворец → дворц`
- [ ] `образец → образц`, `мудрец → мудрец`, `пловец → пловц`

## Этап 3. Сложные фразы

### 3.1 Несколько существительных в фразе

Сейчас «красивый дом и сад» — прилагательное согласуется только с первым существительным.
Нужна группировка: `[прил] + сущ`, союз, `[прил] + сущ`.

**Файл:** `RussianDeclensionEngine.cs` | **Оценка:** ~1–2 дня

- [ ] Разбивать фразу на группы по союзам (`и`, `а`, `но`, `да`, `или`)
- [ ] В каждой группе найти пары прилагательное + существительное
- [ ] Склонять каждую пару с согласованием внутри группы

### 3.2 Прилагательные после существительного

**Файл:** `RussianDeclensionEngine.cs` | **Оценка:** ~полдня

- [ ] Искать прилагательное не только слева, но и справа от существительного
- [ ] Согласование для фраз вида «дом красивый»

### 3.3 Краткие прилагательные

«Хорош», «красив», «велик» — не склоняются по падежам, но анализатор может путать с существительными.

**Файл:** `RussianAnalyzer.cs` | **Оценка:** ~полдня

- [ ] Список употребительных кратких прилагательных: `хорош`, `красив`, `велик`, `могуч`, `свеж`, `горяч`, `богат`, `беден`, `умён`, `глуп`
- [ ] Не склонять краткие прилагательные (возвращать как есть)

### 3.4 Причастия

«Стоящий», «сделанный», «потерянный», «читаемый», «любимый» — склоняются как прилагательные.

**Файл:** `RussianAnalyzer.cs`, `RussianDeclensionEngine.cs` | **Оценка:** ~1 день

- [ ] Распознавать суффиксы действительных причастий: `-ущ/-ющ`, `-ащ/-ящ`
- [ ] Распознавать суффиксы страдательных причастий: `-ем/-им`, `-нн`, `-енн`, `-т`
- [ ] Классифицировать как Adjective для целей склонения
