# Склонятор

**Sclon** — консольное .NET-приложение для склонения русских фраз по всем падежам.

Вводите любую фразу на русском — получаете её во всех шести падежах с вопросами.

## Возможности

- **Склонение существительных** — 1-е (м.р./ср.р.), 2-е (ж.р. на -а/-я), 3-е (ж.р. на -ь) склонения, разносклоняемые (на -мя, путь, дитя)
- **Согласование прилагательных** — прилагательные автоматически согласуются с существительным по роду, числу и одушевлённости
- **Одушевлённость** — винительный падеж различает одушевлённые и неодушевлённые объекты
- **Беглые гласные** — корректное склонение слов с беглой гласной (пёс → пса, день → дня, огонь → огня, камень → камня)
- **Работа с фразами** — предлоги, союзы и знаки препинания сохраняются
- **Определение рода** — обширный словарь исключений для корректного определения рода

## Установка и запуск

### Требования

- [.NET 10 SDK](https://dotnet.microsoft.com/download) или новее

### Запуск

```bash
cd src
dotnet run
```

Программа работает в интерактивном режиме. Введите фразу и нажмите Enter — получите таблицу склонений.

### Примеры

```
Введите фразу (или 'выход' для выхода): красивый дом

  Фраза: "красивый дом"

  Именительный   (кто? что?     ) → красивый дом
  Родительный    (кого? чего?   ) → красивого дома
  Дательный      (кому? чему?   ) → красивому дому
  Винительный    (кого? что?    ) → красивый дом
  Творительный   (кем? чем?     ) → красивым домом
  Предложный     (о ком? о чём? ) → красивом доме
```

## Структура проекта

```
src/
├── Program.cs                        # Точка входа (интерактивная оболочка)
└── Morphology/
    ├── Models.cs                     # Модели данных (падежи, род, часть речи)
    ├── RussianAnalyzer.cs            # Анализ и токенизация русских слов
    └── RussianDeclensionEngine.cs    # Движок склонения
```

### Компоненты

- **Models.cs** — перечисления `RussianCase` (падежи), `GrammaticalGender` (род), `GrammaticalNumber` (число), `Animacy` (одушевлённость), `PartOfSpeech` (часть речи), а также класс `WordInfo` и вспомогательный `CaseHelper`
- **RussianAnalyzer.cs** — токенизация фраз, определение части речи, рода, одушевлённости по окончанию слова, словарь исключений, учёт беглых гласных
- **RussianDeclensionEngine.cs** — правила склонения существительных (3 склонения + разносклоняемые), склонение и согласование прилагательных, склонение фраз целиком

## Ограничения

- Число определяется как единственное (множественное не поддерживается)
- Склонение глаголов, числительных и местоимений не реализовано
- Род для некоторых слов (особенно заимствованных) может определяться неточно
- Фразы со сложной структурой (причастные обороты) обрабатываются упрощённо
